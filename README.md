# 人工智能基础问答系统 (AI Q&A System)

一个基于关键词匹配的人工智能基础知识问答系统，支持可视化交互界面。

## 功能特点

- 预置 18 个 AI 基础问题及答案，覆盖 6 个分类（基础概念、核心算法、工具与语言、应用领域、数据与评估、伦理与社会）
- 可视化图形交互界面（Tkinter GUI），包含标题栏、输入框、回答区、知识库分类浏览栏和状态栏
- 基于关键词集合交集的智能匹配算法
- 支持多次提问，提问历史记录存储于列表中
- 知识库按关键词分类，用于优化查询效率与匹配逻辑

## 运行方式

```bash
python3 ai_qa_system.py
```

## 退出方式

在输入框中输入「退出」并提交即可结束程序，程序会显示所有提问记录。
